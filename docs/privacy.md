# Privacy Policy

*Last updated: 2026-07-13*

## Summary

ese-forensic is a set of local Rust libraries. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

ese-forensic reads only the ESE/EDB files the calling program supplies. All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

ese-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

ese-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/ese-forensic](https://github.com/SecurityRonin/ese-forensic).
